#### Install
go get -u github.com/octoberstorm/zingmp3-dl

#### Usage
zingmp3-dl [options] &lt;Song or Album URL&gt;

#### options
  -dir=&lt;Download_Dir&gt;, default: current directory

### Example
zingmp3-dl http://mp3.zing.vn/bai-hat/My-Love-Westlife/ZWZB70EO.html
